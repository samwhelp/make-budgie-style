## Usage


### Quick Start

> How to download [make-budgie-style.sh](https://github.com/samwhelp/make-budgie-style/blob/main/make-budgie-style.sh) and run [it](https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh).


* run

``` sh
bash <(curl -L https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh)
```


* or run

``` sh
bash <(wget -q https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh -O - )
```


* or run

``` sh
bash -c "$(curl -fsSL https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh || wget -q https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh -O - )"
```


* or run

``` sh
bash -c "$(wget -q https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh -O - || curl -fsSL https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh)"
```


### Basic

> default

``` sh
./make-budgie-style.sh
```


> Option: [REF_MASTER_DISTRO](#option--ref_master_distro)

* execute local script

``` sh
REF_MASTER_DISTRO=archlinux ./make-budgie-style.sh
```

* execute remote script

``` sh
REF_MASTER_DISTRO=archlinux bash <(curl -L https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh)
```


> Option: [REF_MASTER_STYLE](#option--ref_master_style)

* execute local script

``` sh
REF_MASTER_STYLE=layan ./make-budgie-style.sh
```

* execute remote script

``` sh
REF_MASTER_STYLE=layan bash <(curl -L https://raw.githubusercontent.com/samwhelp/make-budgie-style/refs/heads/main/make-budgie-style.sh)
```
